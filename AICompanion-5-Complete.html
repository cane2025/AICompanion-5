🚀 Snabbstart

För att starta projektet:
# Installera dependencies npm install # Starta båda servrarna (API + Frontend) npm run dev:all # API: http://127.0.0.1:3001 # Frontend: http://127.0.0.1:5175 # Login: Använd dev-token "s_demo"
Smoke-test för att verifiera allt fungerar:
# Kör smoke-test npm run smoke # Detta testar: # ✅ API health check # ✅ Login funktionalitet # ✅ Skapa klient # ✅ Skapa vårdplan # ✅ Skapa GFP # ✅ Staff-scoped data access
